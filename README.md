# HaSC
Haskell Small-C Compiler


## 概要
京都大学情報学科計算機コース3回生配当『計算機科学実験及演習3  (ソフトウェア)』の課題リポジトリである．CのサブセットであるSmall CのコンパイラをHaskellで実装する．講義としてはMIPSコードを生成してテストが通れば良いが，実験終了後も拡張を続けていく．
